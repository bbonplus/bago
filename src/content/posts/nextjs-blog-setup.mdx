## 前言

搭建一个个人博客是每个开发者都应该做的事情。今天，我们来看看如何使用现代技术栈快速搭建一个博客。

## 项目结构

一个典型的 Next.js 博客项目结构如下：

```
src/
├── app/           # 页面路由
├── components/    # React 组件
├── content/       # MDX 文章
└── lib/           # 工具函数
```

## 核心功能

### 1. MDX 支持

MDX 让你可以在 Markdown 中使用 React 组件，非常适合写技术博客。

### 2. 静态生成

使用 `generateStaticParams` 在构建时生成所有文章页面，性能极佳。

### 3. 响应式设计

Tailwind CSS 让响应式设计变得简单直观。

## 总结

Next.js + MDX 是搭建个人博客的绝佳选择。它提供了：

- 优秀的开发体验
- 出色的性能
- 灵活的内容管理方式
